# Hardware Components

| Component | Quantity | Description |
| --- | ---: | --- |
| ESP8266 | 1 | Main controller for gas monitoring, LCD display, cloud upload, and GSM alerting |
| ESP32-CAM | 1 | Camera module used for human detection and web interface hosting |
| MQ Gas Sensor | 1 | Gas sensor used to detect harmful gas levels |
| GSM Module | 1 | Sends SMS safety alerts |
| 16x2 I2C LCD | 1 | Displays gas value and system status |
| Jumper Wires | As required | Used for electrical connections |
| Breadboard or PCB | 1 | Used for prototyping or final circuit assembly |
| Power Supply | 1 | Provides stable power to the system |

## Optional Components

- Buzzer for local audible alerts
- Status LEDs
- Enclosure for industrial placement
- Level shifter, if required by the selected modules
